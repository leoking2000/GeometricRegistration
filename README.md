# GeometricRegistration

A C++ project for experimenting with rigid 3D geometric registration algorithms, including ICP (Iterative Closest Point), with optional real-time visualization using raylib.

---

## ✨ Features

- 3D point cloud representation
- Rigid transform estimation (rotation + translation)
- Basic ICP implementation
- RMS error computation
- Real-time visualization (instanced rendering)
- Headless mode support (no window required)

---

## 🧠 Technologies

- C++
- raylib (visualization)
- GLM (math)
- Custom geometric registration core
- CMake
